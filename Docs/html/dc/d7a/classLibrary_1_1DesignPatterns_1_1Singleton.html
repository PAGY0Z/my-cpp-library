Detailed Description

template<typename Type>
class Library::DesignPatterns::Singleton< Type >

Singleton Template Class.

This class template is designed to provide a global point of access to a single instance of a class. The Singleton class template ensures that only one instance of the class is created in a thread-safe manner.

Template Parameters
TypeThe type of the class that will be a Singleton
