typedef enum Library::DesignPatterns::Exceptions::FactoryExceptionsType_e Library::DesignPatterns::Exceptions::FactoryExceptionsType_e

FactoryExceptionsType Enumeration.

The FactoryExceptionsType enumeration defines the types of exceptions that can be thrown by the Factory class template.

Note
The enumeration values are as follows:
- ALREADY_REGISTERED: The key is already registered
- UNKNOWN_KEY_AT_CREATE: The key is unknown at create
- UNKNOWN_ERROR_AT_REGISTER: An unknown error occurred at register
- UNKNOWN_ERROR_AT_CREATE: An unknown error occurred at create
- UNKNOWN_KEY_AT_UNREGISTER: The key is unknown at unregister
- UNKNOWN_ERROR_AT_UNREGISTER: An unknown error occurred at unregister
- UNKNOWN_ERROR_AT_CLEAR: An unknown error occurred at clear

typedef struct Library::DesignPatterns::Exceptions::FactoryExceptionsMessage_s Library::DesignPatterns::Exceptions::FactoryExceptionsMessage_t

FactoryExceptionsMessage Structure.

The FactoryExceptionsMessage structure defines the message associated with each type of exception that can be thrown by the Factory class template.

Note
The structure includes the following members:
- __type: The type of exception
- __message: The message associated with the exception
