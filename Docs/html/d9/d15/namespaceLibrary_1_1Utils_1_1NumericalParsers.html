Check if a string is an integer number.

This function checks if a string is an integer number. If the string is an integer number, the function returns true. Otherwise, the function returns false.

Remarks
A string is considered an integer number if it contains only digits and 0+('-' || '+') at the beginning.
Parameters
stringThe string to check
Returns
