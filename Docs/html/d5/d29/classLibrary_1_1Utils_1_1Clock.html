Detailed Description

Clock Class.

The Clock class is a utility class that provides a way to measure the elapsed time between two points in a program. The class uses the std::chrono library to measure the time elapsed between two points in a program. The class provides methods to start, restart, and get the elapsed time in milliseconds.
